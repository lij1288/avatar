## **HTML的表单标签**

### form

- 表单

- action：接收表单数据的地址

### input

- 输入控件
- type
  - text：单行文本框
  - password：密码框
  - submit：提交按钮
  - button：定义按钮
  - reset：重置按钮
  - radio：单选按钮
  - checkbox：复选按钮
  - file：文件浏览上传
  - select：下拉列表
    - option：选项
  - label：标注
    - for：绑定的表单元素，对应元素的id
- name：元素名称
- value：元素内容
