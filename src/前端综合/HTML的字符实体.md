## **HTML的字符实体**

| 显示 | 描述 | 实体 |
| -------- | ---- | -------- |
| &nbsp;   | 空格 | \&nbsp;  |
| &gt;     | 大于 | \&gt; |
| &lt;     | 小于 | \&lt;    |
| &times;  | 乘号 | \&times; |
| &divide; | 除号 | \&divide; |
| &amp;    | 和号 | \&amp;    |
| &quot;   | 引号 | \&quot;   |
| &apos;   | 撇号 | \&apos;   |
| &copy;   | 版权 | \&copy;   |
