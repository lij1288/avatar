## PotPlayer快捷键

### 隐藏边框

- 皮肤 -> 视频下自动隐藏

### 播放速度

| 功能 | 快捷键 |
| ---- | ------ |
| 加速 | C      |
| 减速 | X      |
| 复原 | Z      |

### 屏幕尺寸

| 功能          | 快捷键      |
| ------------- | ----------- |
| 半倍屏        | 1           |
| 1倍屏         | 2           |
| 全屏          | Alt + Enter |
| 显示器30%尺寸 | Alt + 1     |
| 显示器45%尺寸 | Alt + 2     |
| 显示器60%尺寸 | Alt + 3     |
| 显示器75%尺寸 | Alt + 4     |

### 定位

| 功能   | 快捷键   |
| ------ | -------- |
| 上一帧 | D        |
| 下一帧 | F        |
| 退5秒  | ←        |
| 进5秒  | →        |
| 退30秒 | Ctrl + ← |
| 进30秒 | Ctrl + → |
